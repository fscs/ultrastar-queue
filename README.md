# BA Schaefer Code



## Getting started

- docker compose up
- frontend at: localhost:3000
- backend at: localhost:8000
- backend interactive docs at: localhost:8000/docs or localhost:8000/redoc


- Login for admin
  - username: bestAdmin
  - password: password
